## 2. Местоимения

| **Лицо** | **Ед. число** | **Мн. число**        |
|----------|----------------|----------------------|
| **1 лицо** | Кхар / Ар      | Ррекхар / Ррейар     |
| **2 лицо** | Кехим / Ехи    | Ррекехим / Ррейехи   |
| **3 лицо** | Орос / Ос      | Ррейорос / Ррейос    |

**Указательные местоимения:**

- Ми — это / эта / этот
- Ми + окончание рода:
  - Микош — это (сред. род)
  - Мичир — этот (муж.)
  - Мишчи — эта (жен.) и т.п.

**Признак вежливости:**

- Приставка `ну-` добавляется к местоимению, создавая вежливую форму (аналог "Вы"). Например: нуКехим — вежливое "ты".

**Притяжательные местоимения:**

| **Лицо** | **Форма**      | **Пример**                 |
|----------|----------------|----------------------------|
| 1 л.     | шайар          | шайарышчи дарш (моя вещь)  |
| 2 л.     | шайехи         | шайехичир тама (твой дом)  |
| 3 л.     | шайос          | шайосикош хрома (его тьма) |
| **обобщённо** | **шай**     | шайашчи / шайакош (если важен род) |

> Для создания **множественного числа** используется приставка `рре-`: шай — свой, ррешай — свои

## 3. Родовая система (живое/неживое)

**Родовые окончания:**

- муж. род: `-чир`
- жен. род: `-шчи`
- ср. род: `-кош`
- неопр. род: `-нано`

**Живое/неживое:**

- Живое: `-те`
- Неживое: `-ме`

Примеры:

- Кхарытечир — Я (муж. род, живое)
- Кхарымешчи — Я (жен. род, неживое)

## 4. Глаголы и наклонения

Формы глаголов:

- Инфинитив: `-ур / -иур` — начальная форма (цасдаур — любить)
- Модальная: `-ц` — форма "мочь/хотеть делать"
- Активная: `-и` — основа действия, на которую клеятся окончания

### Спряжение глаголов по лицам и родам (в одной строке):

| Лицо     | Окончания             | Примеры (на основе "цасдаи-")                            |
|----------|------------------------|-----------------------------------------------------------|
| 1 л.     | -р / -и / -ш / -но     | цасдаир / цасдаии / цасдаиш / цасдаино   |
| 2 л.     | -м / -му / -тш / -кс   | цасдаим / цасдаиму / цасдаитш / цасдаикс |
| 3 л.     | -т / -та / -ч / -го    | цасдаит / цасдаита / цасдаич / цасдаиго   |

> Род (по порядку): муж. / жен. / ср. / неопр.

**Сравнение с полной формой:**

| Тип               | Без местоим. | С местоимением        |
|-------------------|--------------|------------------------|
| 1 л. ед. муж.     | цасдаир      | Ар цасдаи              |
| 2 л. ед. жен.     | цасдаиму     | Ехи цасдаи             |
| 3 л. ед. ср.      | цасдаич      | Ос цасдаи              |
| 1 л. мн. муж.     | ррецасдаир   | Ррейар цасдаи          |
| 3 л. мн. неопр.   | ррецасдаиго  | Ррейос цасдаи          |

> Местоимения опциональны, можно использовать или не использовать, зависит от стиля/ясности.

### Повелительное наклонение:

| Форма        | Окончание | Пример         | Значение                 |
|--------------|-----------|----------------|--------------------------|
| Агрессивная  | -цыр      | цасдаицыр ми   | Люби (приказ)            |
| Вежливая     | -цир      | цасдаицир ми   | Люби, пожалуйста         |

Слово "пожалуйста" — `жуле` (используется только с `-цир` для правильной передачи настроения: "цасдаицир ми, жуле")

## 5. Отрицание

- Частица **`ттиз`** — "не", "но"
- Используется между глаголами или для отрицания действия
- Отрицательные постфиксы:
  - `-öл / эчöл` — точно бы
  - `-öз / эчöз` — вряд ли бы
  - `-öк / эчöк` — возможно бы / нейтрально

---

## 6. Падежи

**Список:**

- **Кавартрамм** — основной
- **Эттамм** — дательный (кому/чему): `-эт / -йэт`
- **Зантамм** — творительный (кем/чем): `-зант / -йзант`
- **Туритамм** — местный падеж места/направления: `-ри` + приставка
  - `на-` — где, `ва-` — куда, `ша-` — откуда
  - частички: öш (через), р' (до/к), да (мимо), пти (около), рути (рядом), эфи (напротив), зу'ч (перед), му'ч (после), вве (за)
  - неуточнённые формы: `-лвы / -ылв`
- **Ндантамм** — происхождение/создатель: `-катри` + родовое окончание

## 8. Родовая/уточняющая система

Род можно уточнить через родовое окончание **даже у имён**. Пример:

- Жцынкатричир — божий (в м.р.)
- ттовос Жцынкатричир — сын божий

## 9. Примеры фраз

- **Осичир-öл исуаит, ттиз драйисжемит.**
— Он бы мог, но не стал.
- **Осичир-öз исуаит, ттиз драйисжемит.**
— Он бы вряд ли смог, но не стал.
- **Осичир-öк исуаит, ттиз драйисжемит.**
— Он бы, может, и смог, может и нет, но не стал.
- **Ыдöш хромга кхарыте отоир мулга, ттиз мула шусутнуиго ка кхар.**
— Я ищу свет сквозь тьму, но свет избегает меня.

## 10. Притяжательные местоимения

Формируются через приставку **ша-** + местоимение + родовое окончание:

| Значение | Форма           | Пример                                       |
| -------- | --------------- | -------------------------------------------- |
| мой      | шайарышчи       | шайарышчи дарш — моя вещь                    |
| твой     | шайехичир       | шайехичир тама — твой дом                    |
| его/её   | шайосикош       | шайосикош хрома — его тьма                   |
| свой     | шай + окончание | Шайачир — свой (муж.), Шайашчи — своя и т.п. |

*Можно указать род у местоимения для акцента, но это не обязательно.*

*Для создания множественного числа используется приставка **`рре-`**: шай — свой, ррешай — свои.*
