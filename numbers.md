# Числительные (Rredęjur)

> [!WARNING]
> Система чисел в данный момент нуждается в глобальной переработке!
> Описанная ниже система является лишь примерной и в будущем будет изменена. 
> На данный момент отсюда рекомендуется брать только названия базовых цифр и чисел.

## 1. Базовые числительные

Числительные на Цэккэмаре довольно просты. Они образуются составным способом и пишутся слитно (см. примеры).  
> Для вопроса "Сколько" используются `Тысир` (если можем назвать числом) и `Тысим` (если не можем сказать числом: много/мало и т.п.).

| Категория          | Значение  | Обозначение | Примечание                     |
|:------------------:|----------:|:-------|:------------------------------------|
| **Цифра 0**        | ноль      | [о]    |                                     |
| **Цифра 1**        | один      | [аб’]  |                                     |
| **Цифра 2**        | два       | [ри]   |                                     |
| **Цифра 3**        | три       | [тти]  |                                     |
| **Цифра 4**        | четыре    | [мэр]  |                                     |
| **Цифра 5**        | пять      | [шыс]  |                                     |
| **Цифра 6**        | шесть     | [ал]   |                                     |
| **Цифра 7**        | семь      | [усэ]  |                                     |
| **Цифра 8**        | восемь    | [öб’]  |                                     |
| **Цифра 9**        | девять    | [зор]  |                                     |
| **10-90**          | десятки   | `-ки-` | Вставляется между десятком и числом |
| **100+**           | сотни     | `-цэт` |                                     |
| **1000+**          | тысячи    | `-чар` |                                     |
| **1 000 000+**     | миллионы  | `-лим` |                                     |
| **1 000 000 000+** | миллиарды | `-дар` |                                     |

При использовании обозначения десятков (`-ки-`), необязательно добавлять единицу перед ней, если всё число начинается с 1: `Аб'китти` = `китти` ("13"). Это же касается и остальных разрядов (как "сто тысяч" вместо "одна сто тысяч").

## 2. Примеры составления

| Число         | Правильная запись                          | Расшифровка        |
|---------------|--------------------------------------------|--------------------|
| **10**        | ки                                         | десяток            |
| **17**        | киусэ                                      | 10 + 7             |
| **23**        | рикитти                                    | 20 + 3             |
| **69**        | алкизор                                    | 60 + 9             |
| **100**       | цэт                                        | сотня              |
| **234**       | рицэт-ттикимэр                             | 200 + 30 + 4       |
| **555**       | шысцэт-шыски-шыс                           | 500 + 50 + 5       |
| **824**       | öб’цэт-рики-мэр                            | 800 + 20 + 4       |
| **3000**      | ттичар                                     | 3 тысячи           |
| **1 234 567** | аб’лим.рицэт-ттики-мэр-чар.шысцэт-алки-усэ | лям.234-тыс.567    |
| **9 200 100** | зордар.рицэт-чар.цэт                       | 9млрд.200-тыс.100  |

**Базовая схема построения:**
`[миллиарды].[миллионы].[тысячи].[сотни-десятки-единицы]`
- Внутри каждого разряда (если число составное): ставим дефис `-` (например: `рицэт-ттики-мэр` = `234`).
- Разряды отделяются точкой `.` (например: `...лим.чар.цэт...`).
- `0 (о)` пишется только если нужно явно указать ноль как отдельную цифру, иначе опускается (`1005` = `чар.шыс`).

## 3. Порядковые числительные

Образуются с помощью частицы `вуй`, которая согласуется с существительными.  
> Вопросы: `Вуйлу?` и `Вуйу ...[-ка]?` ("Который?", "Какой по счёту?").
> Для множественного числа используется всё та же приставка индекса `ррэ-`.

**Примеры:**
- `вуй-аб'` — первый
- `ррэвуй-ри` — вторые
- `вуй-ттикитти` — тридцать третий
- `вуй-зорцэт-зорки-зор` — девятьсот девяносто девятый

(Пример: `фа вуйка-шыс сэчпиника` → в пятом классе. `вуйка-шыс` - "в каком? - в пятом").

## 4. Дробные числа

Образуются частицей `йыдтыц` (дословно "в знаменателе").  

**Примеры:**
- `аб' йыдтыц-шыс` — одна пятая
- `тти йыдтыц-мэр` — три четверти
- `усэ йыдтыц-аб'ки` — семь десятых
- `зор йыдтыц-зорки` — девять девяностых

> Для склонения дробных чисел используются вспомогательные слова вместо окончаний или частичек.
